## 技能概述

**技能名称：**  wechat-red-envelope-cover-designer
**核心功能：**  生成符合微信红包封面开放平台规范的封面设计图片，包括封面图、封面挂件、气泡挂件和封面故事素材。
**适用场景：**

- 春节、中秋、情人节、七夕、圣诞、生日、元宵、端午、元旦、国庆等节日主题红包封面设计
- 自定义品牌活动主题红包封面设计
- 企业或个人定制红包封面需求
## 主要功能模块
### 1. 封面图生成（必需素材）
- **尺寸规格：**  957×1278 像素
- **宽高比：**  3:4（竖版）
- **文件大小限制：**  ≤500KB
- **支持格式：**  PNG 或 JPG
- 构图安全区域：
  - 上方25%（约320px）：保持简洁，用于显示用户头像、昵称和祝福语
  - 中下部50%：主要视觉元素区域
  - 底部15%（约190px）：可能被遮挡，避免放置关键元素

### 2. 封面挂件生成（可选素材）

- **尺寸规格：**  1053×1746 像素
- **宽高比：**  约3:5
- **文件大小限制：**  ≤300KB
- **支持格式：**  PNG（必须透明背景）
- 安全区域：
  - 上方75%（约1310px）：设计元素必须位于此区域
  - 下方25%（约436px）：必须保持完全空白/透明（不可编辑区域）

### 3. 气泡挂件生成（可选素材）

- **尺寸规格：**  480×384 像素
- **宽高比：**  5:4
- **文件大小限制：**  ≤300KB
- **支持格式：**  PNG（必须透明背景）
- 安全区域：
  - 上方60%（约230px）：设计元素必须位于此区域
  - 下方40%（约154px）：必须保持空白（预留给"微信红包"文字显示）

### 4. 封面故事图片生成（可选素材）
- **尺寸规格：**  750×1250 像素
- **宽高比：**  3:5
- **文件大小限制：**  ≤300KB（每张）
- **数量限制：**  最多5张
- **支持格式：**  PNG 或 JPG
- 安全区域：
  - 顶部和底部各10%：可能在部分设备上被裁切
  - 中央80%：关键元素集中区域

## 设计流程
### 阶段一：需求确认
在设计开始前，需要确认以下信息：
**基础信息：**
- 封面主题（节日类型或品牌活动）
- 封面简称（最多8个字）
- 是否需要品牌logo
- 目标受众
**设计风格：**
- 整体风格（国潮/简约/插画/3D立体/扁平化/手绘/Q萌可爱/复古等）
- 主色调偏好
- 元素偏好
**素材需求：**
- 封面图（必需）
- 封面挂件（可选）
- 气泡挂件（可选）
- 封面故事图片（可选，最多5张）
### 阶段二：制定设计方案
根据用户确认的需求，展示设计方案和素材清单。
### 阶段三：生成封面图片
按照规范的顺序和尺寸要求生成各类素材。
### 阶段四：交付与审核提醒
提供完整的交付清单和提交审核前的注意事项提醒。
## 支持的节日类型
1. **春节/新年**：生肖主题、传统元素（福字、灯笼、鞭炮、祥云等）
2. **中秋节**：圆月、玉兔、嫦娥、桂花、月饼等
3. **情人节/七夕**：爱心、玫瑰、鹊桥、银河等
4. **圣诞节**：圣诞树、圣诞老人、驯鹿、雪花等
5. **生日**：蛋糕、蜡烛、气球、礼物盒等
6. **元宵节**：花灯、孔明灯、汤圆等
7. **端午节**：粽子、龙舟、艾草、五彩绳等
8. **元旦**：烟花、彩带、倒计时等
9. **国庆节**：烟花、彩旗、祥云等（需注意政治元素限制）
## 审核规范与限制
### 法律法规禁止内容
- 反对宪法基本原则、危害国家统一的内容
- 宣扬恐怖主义、极端主义、邪教、迷信的内容
- 淫秽、色情、赌博、暴力内容
- 毒品、自残相关内容
- 虚假广告、传销、非法集资等
### 平台规范禁止内容
- 二维码、条形码、电话号码、网址等联系方式
- 广告语、促销信息、价格信息
- 未经授权的商标、版权素材、知名IP
- 政治敏感内容（国旗、国徽、天安门等）
- 已被封杀或负面舆论的人物肖像
### 设计规范要求
- 封面图上方25%必须保持简洁，确保白色文字可读
- 所有素材必须严格按照指定尺寸制作
- 文件大小必须在限制范围内
- 挂件必须使用PNG透明背景格式
- 内容适合全年龄段用户
## 交付标准
### 封面图
- 格式：PNG/JPG，≤500KB
- 尺寸：957×1278px
- 质量：清晰锐利，无模糊锯齿
- 内容：不含文字、水印、二维码、联系信息
### 封面挂件
- 格式：PNG透明背景，≤300KB
- 尺寸：1053×1746px
- 要求：元素在上方75%区域，下方25%空白
- 设计：必须有实质性造型，不能是简单线条
### 气泡挂件
- 格式：PNG透明背景，≤300KB
- 尺寸：480×384px
- 要求：元素在上方60%区域，下方40%空白
- 设计：小巧精致，小尺寸下清晰可辨
### 封面故事图片
- 格式：PNG/JPG，≤300KB/张
- 尺寸：750×1250px
- 数量：最多5张
- 内容：具有叙事性，不含广告内容
## 使用建议
1. **风格统一**：所有素材（封面图、挂件、故事图）保持一致的设计语言和风格
2. **色彩协调**：节日主题推荐使用饱和度较高的喜庆色彩
3. **构图合理**：严格遵守安全区域要求，确保关键信息不被遮挡
4. **质量保证**：使用高质量原图，确保清晰度
5. **审核通过**：遵循审核规范，避免使用禁止内容

## references文件清单及用途说明

### 1. dimension-specs.md（尺寸规范文档）

**文档作用：**  定义微信红包封面各类素材的精确尺寸规格和设计要求

### 2. prompt-templates.md（通用提示词模板）

**文档作用：**  提供各类素材的通用提示词模板，确保生成的图片符合微信红包封面规范

### 3. festival-prompts.md（节日专属提示词）

**文档作用：**  为不同节日提供专属的提示词模板，包含节日特定的元素、配色方案和设计要求

### 4. audit-standards.md（审核标准文档）

**文档作用：**  详细说明微信红包封面的审核标准和禁止内容，确保生成的素材能够通过平台审核

## references文件使用原则

1. **优先使用节日专属模板**
   - 当用户选择节日主题时，优先使用 festival-prompts.md 中的对应模板
   - 确保节日元素和配色方案符合节日特色
2. **通用模板作为补充**
   - 当用户有自定义需求或节日专属模板不完全匹配时，使用 prompt-templates.md 中的通用模板
   - 根据用户需求替换变量，补充细节
3. **严格遵守尺寸规范**
   - 所有素材必须严格按照 dimension-specs.md 中的尺寸要求生成
   - 特别注意构图安全区域，确保文字可读性
4. **始终遵循审核标准**
   - 所有设计必须符合 audit-standards.md 中的要求
   - 避免使用任何禁止内容
   - 确保内容适合全年龄段用户
5. **风格保持一致**
   - 所有素材（封面图、挂件、故事图）必须使用统一的设计语言
   - 色彩、风格、元素选择要协调

## 工作流程中的使用

在技能的实际执行过程中，references 文件夹的使用流程如下：

1. **需求确认阶段**
   - 参考 audit-standards.md 确认用户需求不包含禁止内容
   - 根据用户选择的节日类型，定位到 festival-prompts.md 中的对应章节
2. **制定设计方案阶段**
   - 从 dimension-specs.md 中提取各类素材的尺寸信息
   - 从 prompt-templates.md 或 festival-prompts.md 中选择合适的提示词模板
3. **生成封面图片阶段**
   - 使用 dimension-specs.md 中的尺寸规范生成素材
   - 使用 prompt-templates.md 或 festival-prompts.md 中的提示词模板
   - 遵循 audit-standards.md 中的审核标准
4. **交付与审核提醒阶段**
   - 使用 dimension-specs.md 中的文件格式要求进行自检
   - 使用 audit-standards.md 中的自检清单进行最终审核
   - 向用户提供提交审核的注意事项
